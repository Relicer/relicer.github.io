スコア計算基準

HPのみ基準や防御基準はありませんが、対応するキャラが実装されたら追加する予定です。

計算基準 式
攻撃 ATK% + ATK/15 + CR×2 + CD + PEN/17
HP HP% + HP/89 + CR×2 + CD + PEN/17
攻撃のみ ATK%×1.964 + ATK/7.6
防御のみ DEF%×1.226 + DEF/5.9
異常 AP/1.553 + ATK%/1.553 + ATK/23.3 + PEN/26.4
