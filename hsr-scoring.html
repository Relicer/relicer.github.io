スコア計算基準

ヒーラーやサポーターでも活かせるように会心を含まない基準も設定されています

上3つの基準（攻撃/HP/防御）をもとに全ての計算基準で理論値が等しくなるように調整されています
これにより基準をまたいでも評価の比較ができます

計算基準 式
HP HP% + HP/18 + CR×2 + CD
攻撃 ATK% + ATK/9 + CR×2 + CD
防御 DEF%×0.8 + DEF/9 + CR×2 + CD
HPのみ HP%×1.596 + HP/11.278
攻撃のみ ATK%×1.596 + ATK/5.639
防御のみ DEF%×1.277 + DEF/5.639
効果命中 EHR×1.6 + ATK%×1.3 + ATK/6.92
撃破特効 BE×1.004 + CR×2.008 + CD×1.004
撃破特効のみ BE×1.291
会心ダメージのみ CD×1.291
会心のみ CR×2.2 + CD×1.1
